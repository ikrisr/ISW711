# Workshop1 - API REST (Node + Express + MongoDB)


Proyecto de la Semana 1. Se desarrolló una API básica con **Node.js** y **Express**, integrando conexión a **MongoDB Atlas**, con el objetivo de practicar la estructura de una API, el manejo de rutas y las pruebas de endpoints.

---

## Tecnologías utilizadas
- Node.js
- Express
- MongoDB Atlas
- Postman

---

## Instalación
1. Abre la terminal en la carpeta del proyecto.
2. Instala las dependencias:
   ```bash
   npm install

3.Crea un archivo .env en la raíz del proyecto y agrega tu cadena de conexión:
    
DATABASE_URL=tu_string_de_conexion_mongodb

## Ejecución
    ```bash npm start
